## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sehribany/ObiletCase.git
2. Open the project in Xcode:
   ```bash
   cd ObiletCase
   open ObiletCase.xcodeproj
3. Install dependencies:
   ```bash
   pod install
4. Run the project on the simulator or a real device.
